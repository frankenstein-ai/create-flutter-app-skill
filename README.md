# agent-skills

A [Claude Code](https://claude.ai/code) plugin marketplace by [Frankenstein AI](https://github.com/frankenstein-ai).

## Installation

```
/plugin marketplace add frankenstein-ai/agent-skills
```

| Plugin | Description |
